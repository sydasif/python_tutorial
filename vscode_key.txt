Command Palette : ctrl + shift + P
Open Setting : ctrl + ,
shift + alt + f : formate code
Ctrl + p : Open file explorer pane
Ctrl + g : Go to line in current
Ctrl + shift + k : Delete a line + lines
Alt + Click : Select multi line/word to amend

ctrl + / : comment a line / Selected lines
Double Click in explorer pane : Create new folder/file
